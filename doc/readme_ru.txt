Установка платформы Abricos

Для работы системы необходимы следующие хар-ки вашего сайта:

    * PHP не ниже 5 версии
    * MySql не ниже 4 версии
    * Apache версии 2 и выше с загруженными модулями: mod_rewrite 

Для установки движка необходимо сделать следующее:
1) Создайте базу данных на вашем сайте в кодировке utf8 сравнение utf8_general_ci;
2) Извлеките архив в корневой каталог сайта (Важно: текущая версия платформы работает 
только если ее скрипты находятся в корневой папки сайта);
3) Переименуйте файл includes/config.new.php в includes/config.php;
4) Настройте файл config.php для работы с базой данных MySql?:

	$config['Database']['dbname'] = 'имя_базы_данных';
	$config['Server']['username'] = 'имя_пользователя';
	$config['Server']['password'] = 'пароль';

5) Установите права 777 папке cache;
6) Откройте главную страницу вашего сайта в браузере, в этот момент движок создаст все необходимые таблицы в базе данных и откроет главную страницу вашего сайта;
После успешной установки будет создан пользователь admin с правами администратора:

имя: admin 
пароль: admin  

Обязательно смените пароль администратора в панели управления -> пользователи
