= Brick CMS core =
Brick CMS core – это ядро системы управления web-контентом. Основная функциональная обязанность ядра – это четкое управление и сопровождение модулей, которые подключаются к нему. Проще говоря, Brick CMS core – это некоторое подобие небольшой операционной системы на стороне сервера, которая умеет подключать к себе программы (модули) и исполнять их. 

Благодаря архитектуре ядра и относительно гибкому подключению и сопровождению модулей в системе, в Brick CMS можно строить сайты и сервисы практически любой сложности.

== Установка ==
Для работы системы необходимы следующие хар-ки вашего сайта:
 * PHP не ниже 5 версии
 * MySql не ниже 4 версии
 * Apache версии 2 и выше с загруженными модулями: mod_rewrite

Для установки движка необходимо сделать следующее:
 1) Скачайте [http://core.cmsbrick.ru/changeset/latest/trunk/project?old_path=/&format=zip&cust_fn=brickcms&root_rm=trunk/project&root_ad= zip архив] движка Brick CMS;[[BR]]
 2) Создайте базу данных на вашем сайте в кодировке utf8 сравнение utf8_general_ci;[[BR]]
 3) Скачайте последнюю версию системы с официального сайта http://cmsbrick.ru;[[BR]]
 4) Извлеките архив в нужный каталог вашего сайта;[[BR]]
 5) Переименуйте файл includes/config.new.php в includes/config.php  ;[[BR]]
 6) Настройте Brick CMS для работы с базой данных MySql в этом файле (config.php):
{{{	
	$config['Database']['dbname'] = 'имя_базы_данных';
	$config['Server']['username'] = 'имя_пользователя';
	$config['Server']['password'] = 'пароль';
}}}	
 7) Установите права 777 папке temp системы Brick CMS;[[BR]]
 8) Откройте главную страницу вашего сайта в браузере, в этот момент движок установит все таблицы и откроет главную страницу;[[BR]]
После успешной установки будет создан пользователь admin с правами администратора: 
{{{
имя: admin 
пароль: admin  
}}}
Ядро Brick CMS по умолчанию поставляется без модулей, поэтому для полноценной работы вашего сайта необходимо установить необходимые модули
