Пример №1

Данный пример демонстрирует возможности технологии BrickJSEngine, а именно возможность использовать BrickJSComponent, в данном случае простая панель, на любых страницах сайта. При этом нет необходимости следить за тем, какие еще нужны компоненты для работы этого компонента, достаточно подключить его, а он уже в свою очередь подгрузит все необходимое.

Вызвать компонент можно как минимум двумя способами:

  • 1. Статический метод
  • 2. Динамический метод

Статический метод

Данный метод позволяет загрузить необходимый компонент, описав его вызов в кирпиче. Для этого необходимо согласно правилам формирования кирпича установить параметр вызова компонента: 

...
[mjs=example]example1.js[/mjs]
...

тогда при сборке страницы будет добавлен метод загрузки этого компонента.

Просмотреть пример в действии можно здесь.

Динамический метод

Данный метод позволяет вызвать необходимый компонент по требованию предварительно загрузив его и все необходимые компоненты для его работы.

Для того, чтобы открыть тестовую панель диалога, нажмите кнопку

Исходный код: Кнопка "Открыть панель диалога"

Исходный код: Функция ShowExampleSimpleDialog

	function ShowExampleSimpleDialog(){

		// открыть панель "Идет загрузка"
		Brick.widget.WindowWait.show();

		// Вызвать загрузчик компонентов, по окончанию загрузки, открыть панель диалога
		Brick.Loader.add({
			// указать загрузчику загружать компонент example1.js из модуля example
			mod:[{name: 'example', files: ['example1.js']}],
			onSuccess: function() {

				// закрыть панель "Идет загрузка"
				Brick.widget.WindowWait.hide();

				// открыть панель диалога
				new Brick.mod.example.SimpleDialog();			
			}
		});
	};
