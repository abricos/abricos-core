Примеры к платформе Brick CMS

Данный модуль содержит в себе примеры различных задач используемых в разработке модулей для платформы Brick CMS.

В исходном коде модуля затронуты все основные аспекты по разработке модулей, такие как:

  • Использование кирпичей Brick и BrickContent на практике;
  • Технология BrickJSEngine;
  • Разработка страницы для "Панели управления".

В панели управления модуль регистрирует себя в разделе "Разработка - примеры"
